== Description ==

Русские даты. Русские названия месяцев и дней недели для WordPress. Что бы не склонять названия месяцев используйте "--"

== Installation ==

1. Upload the `axelnsk-rus-date` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. No need to configure, it just works!
